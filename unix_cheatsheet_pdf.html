UNIX COMMANDS CHEATSHEET

Essential commands for Unix/Linux systems

FILE & DIRECTORY OPERATIONS

Navigation

pwdPrint working directory
cd [dir]Change directory
cd ~Go to home directory
cd ..Go up one directory
cd -Go to previous directory

Listing

lsList directory contents
ls -lLong format listing
ls -laLong format with hidden files
ls -lhHuman readable file sizes
ls -RRecursive listing

Creating

mkdir [dir]Create directory
mkdir -p [path]Create parent directories
touch [file]Create empty file

Copying & Moving

cp [src] [dst]Copy file
cp -r [src] [dst]Copy directory recursively
mv [src] [dst]Move/rename file
ln -s [src] [link]Create symbolic link

Removing

rm [file]Remove file
rm -r [dir]Remove directory recursively
rm -f [file]Force remove
rmdir [dir]Remove empty directory

FILE VIEWING & EDITING

Viewing

cat [file]Display file content
less [file]View file page by page
more [file]View file page by page
head [file]Show first 10 lines
head -n [num] [file]Show first n lines
tail [file]Show last 10 lines
tail -f [file]Follow file changes

Editing

nano [file]Edit with nano
vim [file]Edit with vim
emacs [file]Edit with emacs

File Information

file [file]Show file type
stat [file]Show file statistics
wc [file]Word, line, character count
wc -l [file]Line count only
du [file/dir]Directory size
du -sh [dir]Human readable size

SEARCHING & FILTERING

Finding Files

find [path] -name [name]Find files by name
find [path] -type fFind files only
find [path] -type dFind directories only
find [path] -size +[size]Find files larger than size
locate [name]Find files by name (fast)
which [command]Find command location
whereis [command]Find command & manual

Text Search

grep [pattern] [file]Search text in file
grep -i [pattern] [file]Case insensitive search
grep -r [pattern] [dir]Recursive search
grep -n [pattern] [file]Show line numbers
grep -v [pattern] [file]Invert match

Text Processing

sort [file]Sort lines
uniq [file]Remove duplicate lines
cut -d',' -f1 [file]Extract columns
awk '{print $1}' [file]Print first column
sed 's/old/new/g' [file]Replace text

PERMISSIONS & OWNERSHIP

Permissions

chmod 755 [file]Change permissions (rwxr-xr-x)
chmod +x [file]Add execute permission
chmod -w [file]Remove write permission
chmod u+r [file]Add read for user
chmod g-w [file]Remove write for group
chmod o+x [file]Add execute for others

Ownership

chown [user] [file]Change owner
chown [user]:[group] [file]Change owner and group
chgrp [group] [file]Change group
chown -R [user] [dir]Change owner recursively

Permission Values

4Read (r)
2Write (w)
1Execute (x)
755rwxr-xr-x
644rw-r--r--
600rw-------

PROCESS MANAGEMENT

Process Information

psShow running processes
ps auxShow all processes
ps -efFull format listing
topDisplay running processes
htopInteractive process viewer
pgrep [name]Find process by name

Process Control

kill [pid]Kill process by PID
kill -9 [pid]Force kill process
killall [name]Kill processes by name
pkill [name]Kill processes by name
nohup [command] &Run command in background

Job Control

jobsList active jobs
bg [job]Put job in background
fg [job]Bring job to foreground
Ctrl+ZSuspend current job
Ctrl+CKill current job

SYSTEM INFORMATION

System Status

uname -aSystem information
whoamiCurrent username
idUser and group IDs
wWho is logged in
uptimeSystem uptime
dateCurrent date and time
calCalendar

System Resources

df -hDisk usage
free -hMemory usage
lscpuCPU information
lsblkBlock devices
lsusbUSB devices
lspciPCI devices

NETWORK COMMANDS

Network Information

ifconfigNetwork interface config
ip addr showShow IP addresses
netstat -anNetwork connections
ss -anSocket statistics
routeRouting table

Network Testing

ping [host]Test connectivity
traceroute [host]Trace route to host
nslookup [host]DNS lookup
dig [host]DNS lookup (detailed)
wget [url]Download file
curl [url]Transfer data from server

ARCHIVE & COMPRESSION

Tar Archives

tar -cvf archive.tar [files]Create tar archive
tar -xvf archive.tarExtract tar archive
tar -czvf archive.tar.gz [files]Create gzipped tar archive
tar -xzvf archive.tar.gzExtract gzipped tar archive
tar -cjvf archive.tar.bz2 [files]Create bzip2 compressed archive
tar -xjvf archive.tar.bz2Extract bzip2 compressed archive
tar -tvf archive.tarList contents of archive

Compression

gzip [file]Compress file
gunzip [file.gz]Decompress file
zip archive.zip [files]Create zip archive
unzip archive.zipExtract zip archive

Input/Output Redirection

command > fileRedirect stdout to file
command >> fileAppend stdout to file
command < fileRead stdin from file
command1 | command2Pipe output to command
command 2>&1Redirect stderr to stdout
command &> fileRedirect both stdout and stderr

Environment Variables

envShow environment variables
echo $VARDisplay variable value
export VAR=valueSet environment variable
unset VARRemove environment variable
alias name='command'Create command alias
unalias nameRemove alias
